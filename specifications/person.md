## Attributes

* ```first_name```

  * description: stores the first names of the individuals
* ```last_name```

* ```energy_level```

    * description: stores how energetic the individuals are
* ```knowledge_level```
    * data type: integer
    * description: stores the knowledge levels of the individuals
